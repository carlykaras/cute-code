Starting PSQL

I had to jump through a few hidden hoops to get PSQL up and running on my Ubuntu WSL. Most sources will insist that Postgres is already installed on Ubuntu but this proved not to be the case for me. Instead I got the most recent package (Version 13 as of writing): apt-get install postgresql-13

Opening PSQL

Setting up Postgresql will create a new 'postgres' user. Switch to that user by running: su postgres Now you're operating as your 'postgres' user and can run postgres. psql If you have a database already set up you can open right to it: psql my_database_name

Creating & Removing Databases

Create a database using 'createdb' and give it a name: createdb new_database_name To remove it you can use 'dropdb': dropdb my_database_name 'Create' and 'Drop' are keywords to recognize throughout PSQL.

Creating & Removing Tables

Now that you have a database it's time to make some tables to store your data in. Open it using: \c my_database_name Here's where we start getting into PSQL commands - as a best practice they're written all caps. You'll use 'CREATE TABLE' to make them, but you'll need to add parameters. CREATE TABLE first_table ( End that first line with an open paranthesis - PSQL won't run anything until you end with a semicolon. first_column text, Columns are made up of two items: column name and data type. second_column integer Then close it out. ); Just like databases, if you want to remove a table, you DROP it. DROP TABLE first_table;

Adding Data to Your Tables

Now you can start adding your data! We'll go over the simple manual process. 'INSERT INTO' is our keyword for inserting data into a table. INSERT INTO first_table VALUES ('Text goes here', 0); The 'VALUES' keyword specifies what exactly we're inserting. You have to enter exactly what the table is expecting, so in our case we need something for first_column that is text and second_column that is an integer.

Updating Data

Sometimes you add columns after you've started to put data in. To update an existing table use ALTER TABLE. To add a new column: ALTER TABLE first_table ADD COLUMN name text; To update an existing row for this new column, you'll have to be specific and tell PSQL exactly which row you want to change. UPDATE first_table SET name = 'Okoye' WHERE second_value = 0;

Queries - How to see your data

You can run a Query on your table to see how your data is going in. SELECT * FROM first_table;
